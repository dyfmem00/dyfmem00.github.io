# 리눅스 사용법 (how to use linux in vb,　vbでのlinuxの使い方)

#### Shell 셸 
- 유저가 리눅스 시스템을 이용할 수 있는 인터페이스 
> 유저 -> 셸 -> 커널 

##### 셸 사용법 
**셸 프롬프트**

- id 명령어
> id: 현재 유저의 유저id, 해당 유저가 속해 있는 그룹 id를 보여주는 명령어 

- sudo apt update
> apt 명령어로 설치 가능한 소프트웨어 패키지 목록을 업데이트
> 각 소프트웨어 패키지 별로 어떤 버전을 설치할 수 있는지도 업데이트 

- sudo apt upgrage
> 소프트웨어 패키지의 버전을 업그래이드 

- pwd (Print Working Directory)
> 현재 작업 중인 디렉토리의 경로 출력

- ls, ls -l 
> 

- cd (Change Directory)
>

절대 경로 상대 경로 

- mkdir (Make Directory)
> 디렉토리 생성하는 명령어 

- touch
> 비어 있는 새로운 파일을 만드는 데 사용

- mv (Move)
> 파일, 디렉토리의 위치를 옮길 때 사용하는 명령어
> 파일, 디렉토리의 이름을 변경할 때 사용할 수 도 있다.

- rm (Remove)
> 파일, 디렉토리를 삭제 하는 명령어

- cat 
> 파일의 내용을 출력하는 명령어

- file
> 파일의 유형을 출력하는 명령어

- echo 
> 셸이 유저가 입력한 텍스트 출력
> echo "입력한 문장" > 폴더이름 ----> 입력한 내용을 파일 내용으로 저장

- cp(Copy)
> 줄임말로 파일이나 디렉토리를 복사하는 명령어
